== Description ==

You can add shortcode you can use at anywhere in your articles.
Thant is simple and useful.

You also place at specific place in your website, such as before content, after content, widget.


== Installation ==

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'advanced ads'
3. Click 'Install Now'
4. Activate Zumi Ad Shortcode on the Plugin dashboard


== Frequently Asked Questions == 

= How to make a new ad? =

1. Go to [Manage Ads] and click [Make New]

2. Put any Ad ID and Code that you want to show.

3. Click [Save Changes]

= How to use ad shortcode? =

You can use short code like this.
[ad {ad_id}]
then, it shows up in your article.

= How to add an add to widget? =

1. Go to Widget on Wordpress sidebar.

2. Add a widget that named Zumi Ad Shortcode.

3. Select your ads from the pull down menu.

*You need to add a fixed ad before adding to widget.
